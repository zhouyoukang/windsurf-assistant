# Windsurf小助手

> 无感号池引擎 · 即时切换 · 零中断

## 安装

1. 下载 [最新 VSIX](https://github.com/zhouyoukang/windsurf-assistant/releases/latest)
2. `Ctrl+Shift+P` → `Extensions: Install from VSIX...` → 选择下载的 `.vsix`
3. `Ctrl+Shift+P` → `Reload Window`

## 使用

- 侧边栏 **Windsurf小助手** 图标 → 管理面板
- `Ctrl+Shift+P` → 输入 `Windsurf小助手` 查看所有命令
